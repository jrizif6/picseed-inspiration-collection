## PicSeed 核心功能拆解

### 1. 一键收藏，不止复制链接

收藏方式有三种：

1. **系统分享菜单**：在任意 App 里点"分享"，选择 PicSeed，内容直接进入收藏库
2. **快捷指令**：在「设置」里添加 PicSeed 快捷指令，一步操作完成收藏，比打开 App 更快
3. **剪贴板识别**：复制链接后打开 PicSeed，App 会自动识别剪贴板内容并提示解析

三种方式覆盖了不同的使用习惯。刷得快的人，用快捷指令最顺手；不习惯配置的人，分享菜单就够了。

### 2. 智能分类 + 自定义标签

这是 PicSeed 最省心的地方。收进来的内容，会**自动按来源平台打标签**——从小红书进来的自动标"小红书"，从 Instagram 进来的标"Instagram"。

然后你还可以自己加自定义标签，比如"配色参考""穿搭灵感""做菜教程"。标签可以多选，一条内容打三四个标都行。

翻找的时候，按标签筛选，几秒钟定位。

### 3. 收藏笔记——灵感不丢失

很多时候，存一张图，是因为当时有个想法。但想法不记下来，两周后你根本不记得当时在想什么。

PicSeed 支持给每条收藏加**笔记**，随时记录想法、分析或备注。Pro 版还支持编辑功能——可以直接在收藏内容上删掉无用的文案，只留下你真正关心的部分。

### 4. 桌面小组件——灵感触手可及

这个功能设计得挺有意思：你可以把收藏的内容加到 iPhone 或 iPad 的桌面小组件里，App 会自动匹配布局，让你每次解锁手机都能看到自己攒的灵感。

对设计师、博主来说，这相当于把「参考板」直接放到桌面上了。

### 5. 画廊视图 + 瀑布流浏览

Pro 版解锁画廊模式，主页切换成纯图片瀑布流展示。如果你主要收藏的是视觉类内容（穿搭、设计、摄影），这个模式看起来比列表顺眼很多——就像带着一个私人壁纸库。

### 6. 跨设备同步，iCloud 保障

iPhone 上存的内容，Mac 上同步可见，iPad 上也能查。用 iCloud 做数据同步，不需要额外注册账号。
